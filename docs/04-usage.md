## 🔍 マッチ検出

- WoWSでマッチが始まると**自動的に検出**されます
- リプレイファイルを読み取り、各APIからそのマッチのプレイヤーの戦績を取得します

## 📊 統計画面

味方・敵の2チームに分かれたテーブル表示で各プレイヤーの戦績が確認できます。

| 表示項目 | 説明 |
| --- | --- |
| マッチ情報 | マップ名、戦闘タイプ、時刻 |
| プレイヤー戦績 | 各チーム・プレイヤーの統計テーブル |
| チーム比較 | PR、ダメージ、勝率、戦闘数の平均比較グラフ |

## 👤 プレイヤー詳細

プレイヤー名の左のボタンを**クリック**すると詳細統計へのリンクがモーダルで表示されます。

## ⚓ 艦艇詳細

艦艇名の左のボタンを**クリック**すると艦艇別の統計へのリンクが表示されます。

## ⚛️ 詳細設定

TonangoRadarでは様々な統計データを表示できるようになっているため、自分好みにカスタマイズしましょう！
