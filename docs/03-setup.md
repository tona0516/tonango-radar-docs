アプリを初めて起動したら、以下の設定を行います。

## 1. ゲームクライアントパスの指定

- 設定画面から`WorldOfWarships.exe`があるフォルダを選択してください
- アプリはこのパスのリプレイファイルを読み取ってマッチを検出します

## 2. サーバー/リージョンの選択

- **EU** / **NA** / **Asia** から自分のアカウントが属するサーバーを選択してください
- デフォルトはAsiaです
- 変更は次のマッチから反映されます

<div class="note" markdown="1">
サーバー設定を変更した場合、反映されるのは次のマッチからです。現在のマッチには影響しません。
</div>
