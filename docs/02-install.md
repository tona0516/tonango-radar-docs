## 💻 動作環境

| 項目 | 要件 |
|------|------|
| OS | Windows 11 x64 |
| World of Warships クライアント | Game Center もしくは Steam 経由でのインストール |
| その他 | インターネット接続 |

## 🚀 インストール手順

1. [ダウンロードページ（Google Drive）](https://drive.google.com/drive/folders/1CoVadYdK-zhtJcM2-O9qBCmRSSbC_dLm?usp=sharing){:target="_blank"}を開き、最新版のインストーラをダウンロードします
2. ダウンロードしたインストーラを実行します
3. 画面の指示に従いインストールを完了してください
4. インストール完了後、デスクトップにショートカットが作成されます

![TonangoRadar ショートカット]({{ '/shortcut.png' | relative_url }})

---

インストールが完了したら、次は[⚙️ 初期設定]({{ site.baseurl }}/setup/)を行いましょう。
